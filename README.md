## whoami

I write code, automate things that should not have been manual,
and occasionally solder something that was not supposed to be soldered.
My interests sit at the intersection of software, hardware, and the
quiet satisfaction of watching a script you wrote at 2am still
running cleanly six months later.

Currently somewhere between a backend, a breadboard, and a Grafana dashboard.
